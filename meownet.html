Meownet

Meownet is what I call my local network./p>

Nodes:

  • VISIONS, an ESA/390 mainframe running VM/ESA 2.4.
  • ITOMORI, a System Z mainframe running z/VM 6.3.
  • TOGETOGE, a Raspberry Pi 4 Model B (2GB), running Debian 12.
  • RENO, a PC running Windows 10 Enterprise N LTSC 21H2.

Notes:

All mainframes are emulated under Hercules on TOGETOGE.

TOGETOGE used to run FreeBSD 13.3, but Hercules's tun/tap networking does not function correctly on FreeBSD, so I begrudgingly settled for debian.

RENO runs the N version of Enterprise LTSC because I thought it only excluded media player, and didn't realize it excluded codecs too. (However these can be reinstalled.)

Names:

  • VISIONS is named after the Red Vox album Visions.
  • ITOMORI is named after the lake in the anime Your Name.
  • TOGETOGE is named after the band in the anime Girls Band Cry.
  • RENO is named after the Red Vox song Reno.

Networking

All machines on MEOWNET run TCP/IP networking. Additionally, both mainframes run NJE.

RENO and TOGETOGE have NJE-II installed.

VISIONS runs the Enterprise Web/VM web server.

TOGETOGE runs FTP and NFS.
