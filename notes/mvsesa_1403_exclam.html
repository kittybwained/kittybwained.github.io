On MVS/ESA 5.2.2, if you try and print something containing an exclamation mark (or cent sign, broken pipe, or backtick), it will be replaced by a space.
To fix this, you need to set TRANS=NO on your printer in JES2. Either edit SYS1.PARMLIB(JES2PARM) and add it to the PRT(1) definition, or run $T PRINTER(1),TRANS=NO in the MVS system console.
